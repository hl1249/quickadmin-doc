# 布局

## 整体布局

采用经典的**左右布局**：

`
+------------------------------------------------+
|  侧边栏 (sidebar)   |        主区域              |
|   Logo              |  顶部导航栏 (header)        |
|   菜单树            |  Tags 标签栏               |
|   收缩按钮          |  页面内容 (main)           |
|                     |  页脚 (footer)             |
+------------------------------------------------+
`

## 布局组件

位于 src/layout/：

| 组件 | 说明 |
|------|------|
| Layout.vue | 主布局容器 |
| Sidebar.vue | 侧边栏（菜单树） |
| Header.vue | 顶部导航 |
| Tags.vue | 多标签页 |
| Main.vue | 内容区域 |
| Footer.vue | 页脚 |

## 主题

- 支持亮色/暗黑模式切换
- 支持侧边栏展开/收缩
- 支持多级菜单折叠
